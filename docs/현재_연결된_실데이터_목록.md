# 현재 연결된 실데이터 목록

최종 업데이트: 2026년 6월 3일

이 문서는 현재 코드에서 실제로 연결된 데이터만 정리한 문서입니다.
더미 데이터이거나 아직 연결되지 않은 항목은 하단에 별도로 적었습니다.

## 1. Module 1 시황 브리핑

### 1-1. 코스피 / 코스닥 지수
- 데이터: 코스피/코스닥 지수, 등락률, 거래대금, 전일대비
- 출처: 네이버 증권
- 코드: src/collectors/market/krx_collector.py

### 1-2. 거래대금 상위 종목
- 데이터: 종목명, 현재가, 등락률, 거래대금
- 출처: 네이버 증권 거래상위 페이지
- 코드: src/collectors/market/naver_market_collector.py

### 1-3. 외국인 / 기관 / 개인 수급 요약
- 데이터: 외국인/기관/개인 순매수 총액 (코스피/코스닥 시장별 포함)
- 출처: 네이버 증권 시세 메인
- 코드: src/collectors/market/naver_market_collector.py

### 1-4. 외국인 / 기관 순매수·순매도 상위 종목
- 데이터: 외국인/기관 각 순매수 상위 5종목, 순매도 상위 5종목
- 출처: 네이버 증권 시세 메인 (frgn_deal_tab, organ_deal_tab)
- 코드: src/collectors/market/naver_market_collector.py

### 1-5. 프로그램 매매
- 데이터: 차익 / 비차익 / 전체 (코스피/코스닥 시장별 포함)
- 출처: 네이버 증권 시세 메인
- 코드: src/collectors/market/naver_market_collector.py

### 1-6. 상한가 종목
- 데이터: 코스피 + 코스닥 상한가 종목 합산
- 출처: 네이버 증권 상한가 페이지
- 코드: src/collectors/market/naver_market_collector.py

### 1-7. 글로벌 / 매크로
- 데이터: 다우, S&P500, 나스닥, 달러/원, 미국 10년물, WTI, 중국 상해, 중국 심천
- 출처: FinanceDataReader (SSEC, SZSC 등)
- 비고: 심천(SZSC)은 데이터 없을 경우 None 반환
- 코드: src/collectors/market/global_collector.py

### 1-8. 시장 주요 뉴스 ✅ 신규 추가
- 데이터: 주요 시황 뉴스 제목, 언론사, 작성 시간, 원문 링크 (최대 12건)
- 출처: 네이버 증권 주요뉴스(mainnews.naver)
- 제공 형태: TXT에는 `주요 뉴스` 섹션의 제목+링크, HTML 대시보드에는 `LATEST NEWS` 링크 카드
- 코드: src/collectors/market/naver_market_collector.py

### 1-9. 업종별 등락
- 데이터: 업종명, 등락률 (최대 20개 업종)
- 출처: 네이버 증권 sise_group.naver
- 비고: 컬럼 파싱이 추정 기반이라 페이지 구조 변경 시 깨질 수 있음
- 코드: src/collectors/market/naver_market_collector.py

### 1-10. 당일 상승 / 하락 상위
- 데이터: 코스피 + 코스닥 당일 상승/하락 상위 종목명 + 등락률 (상승 최대 50종목, 하락 최대 20종목)
- 출처: 네이버 증권 상승/하락 페이지
- 비고: 기존 52주 신고가/신저가 URL이 폐기되어 당일 상승/하락 상위로 대체
- 코드: src/collectors/market/naver_market_collector.py

### 1-11. 5% 이상 상승 종목
- 데이터: 등락률 상위 중 +5% 이상 상승한 종목의 시장, 종목명, 종목코드, 현재가, 등락률
- 출처: 네이버 증권 상승 페이지 (코스피 + 코스닥)
- 코드: src/collectors/market/naver_market_collector.py

### 1-12. 시간외 단일가 급등락 (부분 연결)
- 데이터: 시간외 단일가 급등락 종목명 + 등락률 (최대 10종목)
- 출처: 네이버 증권 NXT 상승/하락 페이지
- 비고: 등락률 파싱이 페이지 구조에 따라 불완전할 수 있음
- 코드: src/collectors/market/naver_market_collector.py

## 2. Module 3 개별 종목 분석

### 2-1. 기본 시세 / 밸류에이션
- 데이터: 현재가, 등락률, 거래대금, 시가총액, 52주 고저, PER, PBR, ROE
- 출처: FinanceDataReader + 네이버 증권 종목 메인
- 코드: src/collectors/stock/naver_stock_collector.py

### 2-2. 이동평균선 위치
- 데이터: 5일/20일/60일선 대비 현재가 위치 (상회/하회/근접 + 괴리율)
- 출처: FinanceDataReader (최근 160일 종가)
- 코드: src/collectors/stock/naver_stock_collector.py

### 2-3. 외국인 / 기관 최근 20일 수급
- 데이터: 외국인/기관 최근 20거래일 누적 순매수
- 출처: 네이버 증권 frgn.naver
- 비고: 주 수 기준 (금액 기준 아님) — AI 입력 시 이 점 명시 필요
- 코드: src/collectors/stock/naver_stock_collector.py

### 2-4. 최근 뉴스
- 데이터: 종목 뉴스 헤드라인 6건, 언론사, 작성 시간, 원문 링크
- 출처: 네이버 금융 종목 뉴스 탭
- 제공 형태: TXT에는 제목/언론사/시간/원문 링크, HTML 대시보드에는 원문 링크 카드
- 코드: src/collectors/stock/naver_stock_collector.py

### 2-5. 증권사 리포트
- 데이터: 리포트 제목, 증권사명, 투자의견, 목표가, 작성일, 요약문
- 출처 1: 네이버 증권 리서치 (company_list.naver)
- 출처 2: fnguide (SVD_Report_Summary_Data) — 요약문 포함
- 코드: src/collectors/stock/naver_stock_collector.py, fnguide_report_collector.py

### 2-6. 컨센서스 목표가
- 데이터: 목표주가, 컨센서스 투자의견
- 출처: 네이버 증권 frgn.naver (실패 시 리포트 목표가 평균으로 보완)
- 코드: src/collectors/stock/short_selling_collector.py

### 2-7. 공매도 거래량 비중
- 데이터: 최근 공매도 거래량 비중
- 출처: KIS Open API `daily-short-sale` (국내주식 공매도 일별추이)
- 비고: 개별 종목 공매도 잔고 비율은 검증된 KIS 필드가 없어 `—`로만 표시
- 코드: src/collectors/stock/kis_stock_collector.py

### 2-8. 최근 공시
- 데이터: 종목 기준 최근 45일 공시 목록 (최대 12건)
- 출처: DART OpenAPI
- 코드: src/collectors/stock/dart_stock_collector.py

### 2-9. 최근 재무 요약
- 데이터: 최근 4개 보고서 기준 매출 / 영업이익 / 순이익
- 출처: DART OpenAPI (fnlttSinglAcnt)
- 코드: src/collectors/stock/dart_stock_collector.py

### 2-10. 대주주 지분율
- 데이터: 최신 대주주 지분율, 대표 보유자명
- 출처: DART OpenAPI (majorstock, elestock)
- 코드: src/collectors/stock/dart_stock_collector.py

### 2-11. 재무 위험 체크
- 데이터: CB / BW / 유상증자 공시 키워드 감지, 위험 플래그 목록
- 출처: DART OpenAPI 공시 텍스트 키워드 검색
- 비고: 정량 지표(부채비율 등)는 미포함
- 코드: src/collectors/stock/dart_stock_collector.py
