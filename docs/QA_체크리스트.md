# QA 검증 체크리스트

배포본을 직접 클릭하며 하나씩 확인하는 시나리오.
각 항목: **할 일 → 정상이면 이렇게 보임**. 이상하면 "어느 항목/무슨 화면"인지 메모해 주세요.

---

## 0. 기본 진입
- [ ] 앱 접속 → 상단에 고양이 로고 + "CAT STOCK" 보임
- [ ] 로고 클릭 → "CAT COIN"으로 바뀌고 탭도 코인용으로 교체
- [ ] 다시 클릭 → "CAT STOCK"으로 복귀
- [ ] 폰으로 접속해도 레이아웃 안 깨짐

---

## 1. 시황 브리핑 (STOCK)
- [ ] "시황 브리핑" 탭 → 기준일 오늘로 두고 "브리핑 생성 →" 클릭
- [ ] 결과 영역에 **TXT 박스가 바로** 보임
- [ ] TXT 박스에서 전문 스크롤/드래그/복사 버튼 동작
- [ ] "신문 보기 →" 클릭 → 신문 대시보드(KOSPI DISPATCH) 전용 화면으로 이동
- [ ] "← 텍스트 화면으로 돌아가기" 클릭 → 기존 입력/텍스트 화면으로 복귀
- [ ] 상단 티커: 코스피/코스닥/달러원/나스닥 숫자 채워짐
- [ ] 차트(거래대금 상위, 수급) 보임
- [ ] **그래프 위에서 화면 아래로 스크롤** → 멈추거나 쉭쉭 안 바뀌고 정상으로 내려감 (특히 폰)
- [ ] 가로 스크롤 안 생김 (차트가 화면 밖으로 안 넘침)
- [ ] "LATEST NEWS" 뉴스 제목 보이고 **클릭 시 기사로 이동**
- [ ] "ANALYST REPORTS" 증권사 리포트 제목 보이고 **클릭 시 리포트로 이동**
- [ ] 외국인/기관 수급 숫자 채워짐 (순매수/순매도 상위 종목 나옴)
- [ ] **선물 수급 / "미제공" / 공매도 잔고 같은 빈 줄이 안 보임** (노이즈 제거 확인)
- [ ] 큰 금액이 `12.2조`처럼 조 단위로 깔끔 (`121726.1억` 아님)
- [ ] 주말/휴장일 기준일을 고르면 최근 거래일 기준 안내가 보임
- [ ] "TXT 다운로드" / "신문 HTML" 다운로드 동작

## 2. 개별 종목 분석
- [ ] "개별 종목 분석" 탭 → 종목명 "삼성전자" 입력, 리포트 기간 최근 1개월로
- [ ] "분석 생성 →" → 결과 영역에 TXT 박스가 바로 보임
- [ ] TXT 박스에서 전문 스크롤/드래그/복사 버튼 동작
- [ ] "신문 보기 →" 클릭 → 종목 신문 전용 화면으로 이동
- [ ] "← 텍스트 화면으로 돌아가기" 클릭 → 기존 입력/텍스트 화면으로 복귀
- [ ] 기본 정보(현재가/시총/PER 등) 채워짐
- [ ] 수급: 외국인/기관 20일·당일 **금액 기준**, 공매도 **거래량 비중** 보임
- [ ] **공매도 "잔고 비율" 줄은 안 보임** (제거 확인)
- [ ] 차트(재무, 수급) 스크롤 정상 (위 1번과 동일 체크)
- [ ] 최근 뉴스 제목+링크 / 증권사 리포트(네이버+fnguide) 나옴
- [ ] 리포트 기간을 오늘 하루로 바꾸면 fnguide는 적게/없을 수 있음 (정상 — 기간 탓)

## 3. 테마 공부
- [ ] "테마 공부" 탭 → "반도체" 입력 → "자료 생성 →"
- [ ] 관련 종목(시총/현재가/등락률/PER) 나옴, `None`·이상한 공백 없음
- [ ] 관련 뉴스 제목+링크 나옴
- [ ] **글로벌 피어 섹션이 안 보임** (제거 확인)
- [ ] "신문 보기 →" 클릭 → 테마 신문 전용 화면으로 이동

## 4. CAT COIN 모드
- [ ] 로고로 COIN 전환 → "처음 가이드" 탭에 용어 설명 카드 보임
- [ ] "코인 시황" → "시황 불러오기" → 대시보드(BTC/ETH/도미넌스/공포탐욕/업비트) 보임
- [ ] 코인 차트도 스크롤 정상 (그래프 위에서 안 멈춤)
- [ ] "개별 코인" → "비트코인" 입력 → 분석 → 가격/시총/FDV 등 보임, `None` 노출 없음
- [ ] "섹터 공부" → "섹터 불러오기" → 섹터별 시총/대표코인 보임
- [ ] "학습 노트" → 입력 후 "노트 만들기" → 정리 텍스트 나옴

---

## 5. AI 칼럼 (OPENAI_API_KEY 설정 시)
- [ ] 시황/종목 대시보드 상단 "MARKET INSIGHT" 칼럼 — **제목 크게**(### 같은 기호 없음), 본문 자연스러움
- [ ] 키 없으면 "OPENAI_API_KEY가 설정되지 않았습니다" 안내 (에러 아님)
