# 작업 지시: 주식 모드 출력영역 — 대시보드를 기본 표시로

발행: 관리자 / 수행: 개발자
브랜치: develop (main 직접 작업 금지)
근거: 주인 지시 — "대시보드 띄우는 게 기본"

## 현황
- 주식 시황/종목/테마: 출력영역(out_col)에 **TXT 터미널 박스가 메인**,
  대시보드는 하단 "대시보드 미리보기" expander(접힘)에 있음.
- 코인: 출력영역에 **대시보드(st.html)가 바로 메인**. (이게 목표 형태)

## 요구 결과
주식 3개 페이지(`_render_market_page`, `_render_stock_page`, `_render_theme_page`)의
출력영역을 코인과 동일 패턴으로:
- **대시보드(build_*_dashboard)를 out_col 메인으로 바로 렌더** (`st.html`).
- 결과 없을 때는 기존 빈 상태 안내(placeholder).
- "대시보드 미리보기" expander는 제거(메인이 됐으므로 중복).

## TXT 접근성 유지 (중요)
TXT는 "유저가 AI에 붙여넣는" 핵심 산출물이라 없애지 말 것.
- 기존 **TXT 다운로드 버튼 유지**(컨트롤 영역).
- 화면 복사용 TXT 박스는 **"텍스트로 보기/복사" expander로 이동**해 대시보드 아래 보조 배치.
  (코인의 "학습 노트 TXT" 다운로드처럼, 복사 기능은 살리되 메인은 대시보드)

## 검증
- 시황/종목/테마 생성 시 대시보드가 바로 보이는지.
- TXT 복사/다운로드가 여전히 동작하는지.
- 로컬 `streamlit run`으로 3개 탭 + 코인까지 렌더 확인.
- `.venv/bin/python3 -m pytest` 전체 통과.

## 주의
- main 푸시 금지. develop 커밋만.
- 코인 모드는 이미 대시보드 메인이므로 건드리지 말 것.
