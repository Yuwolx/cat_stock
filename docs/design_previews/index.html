🐱 CAT STOCK 디자인 후보 3종

awesome-design-md 레포(74개 브랜드)에서 서비스 성격에 맞춰 추린 후보. 각 카드를 클릭하면 같은 시황 브리핑 데이터를 해당 스타일로 렌더링한 목업이 열립니다. 아직 앱에는 적용하지 않았습니다.

1. Wired — 정제된 흑백 에디토리얼 신문 컨셉의 진화

흑백 듀엣 + 세리프 디스플레이 + 헤어라인 + 사각 지오메트리. 장식을 다 걷어내고 타이포그래피가 전부인 잡지 문법.

어울리는 이유: 지금의 "신문(DISPATCH)" 정체성을 버리지 않고 세련되게 다듬는 방향. 데이터 밀도가 높아도 시끄럽지 않음. 리스크: 무채색이라 "선물" 특유의 따뜻함은 약해짐.

2. PostHog — 크림 캔버스 + 고양이 마지널리아 선물 감성 최대

따뜻한 크림 배경, 흰 카드 + 올리브 보더, 옐로 CTA, 그리고 페이지 곳곳의 마스코트. PostHog의 고슴도치 자리를 고양이가 대신함.

어울리는 이유: cat_stock의 고양이 정체성과 "여자친구 선물"이라는 본질에 가장 밀착. 데이터 카드는 진지하고 캐릭터는 다정한 대비가 이 서비스의 목소리와 같음. 리스크: 금융 서비스로서의 긴장감은 셋 중 가장 낮음.

3. Stripe — 핀테크 인프라 무드 프로덕트 신뢰감

네이비 잉크 + 일렉트릭 인디고, 상단 그라디언트 메시, 얇은 웨이트 타이포, tabular 숫자, 필 버튼. 돈을 다루는 서비스의 표준 문법.

어울리는 이유: "다른 유저가 생길 수도 있는" 프로덕트로 키울 때 가장 확장성 있는 얼굴. 숫자 가독성(tabular figures)이 금융 데이터와 찰떡. 리스크: 개성(신문·고양이)이 가장 많이 희석됨.
