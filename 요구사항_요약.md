# 주식 AI 분석 프로그램 요구사항 요약

작성일 기준 원문: 2026년 5월 28일

## 1. 이 프로그램의 목적

이 프로그램은 한국 주식시장 데이터를 자동으로 모아서,
AI(ChatGPT / Gemini)에 바로 붙여넣을 수 있는 텍스트 파일을 만들어주는 도구입니다.

핵심 목적은 아래 3가지입니다.

1. 매일 장 마감 후 시황을 자동 정리한다.
2. 특정 테마를 공부할 때 필요한 자료를 한 번에 모은다.
3. 개별 종목 분석용 자료를 기관 리서치처럼 정리한다.

중요한 점:

- AI API를 따로 붙이지 않음
- 추가 API 비용 없이 사용
- 결과물은 `.txt` 파일
- 사용자는 그 파일 내용을 복사해서 ChatGPT/Gemini 채팅창에 붙여넣으면 됨

---

## 2. 전체 동작 방식

프로그램의 흐름은 아주 단순합니다.

1. 프로그램이 자동 또는 수동으로 실행된다.
2. 웹/API에서 필요한 데이터를 수집한다.
3. 수집한 데이터를 정리해서 텍스트 파일로 저장한다.
4. 사용자가 파일을 열어 내용을 복사한다.
5. ChatGPT Plus 또는 Gemini Advanced에 붙여넣는다.
6. AI가 최종 분석을 해준다.

즉, 이 프로그램의 역할은 "분석"이 아니라
"AI가 분석하기 좋은 재료를 자동으로 모아주는 것"입니다.

---

## 3. 프로그램 구성

프로그램은 총 3개 모듈로 구성됩니다.

### Module 1. 시황 브리핑 자동화

매일 장 마감 후 한국 증시 전체 상황을 자동 수집해서 정리하는 기능입니다.

수집 대상 예시:

- 코스피 / 코스닥 등락률, 거래대금
- 업종별 등락
- 거래대금 상위 종목
- 외국인 / 기관 / 개인 수급
- 외국인 / 기관 순매수·순매도 상위 종목
- 선물 / 옵션 / 프로그램 매매
- 52주 신고가 / 신저가
- 상한가 종목
- 시간외 단일가 급등락
- 당일 주요 공시
- 증권사 리포트
- 미국 지수, 환율, 미국 10년물 금리, 유가, 중국 증시

출력 결과:

- `output/시황브리핑_YYYYMMDD.txt`
- 1차 버전에서는 데이터 결과만 정리해서 제공
- 추후 원하면 시황 분석 프롬프트 자동 첨부 기능 추가 가능

실행 방식:

- 매일 오후 3시 30분 이후 자동 실행
- 필요하면 버튼으로 수동 실행 가능

이 모듈이 1순위 핵심입니다.

### Module 2. 테마 공부

특정 테마를 입력하면, 그 테마 공부에 필요한 자료를 자동으로 모아주는 기능입니다.

수집 대상 예시:

- 테마 관련 종목 리스트
- 각 종목 기본 정보
- 최근 2주 뉴스
- 관련 공시
- 증권사 리포트
- 글로벌 피어 종목 정보

출력 결과:

- 테마명 기준 텍스트 파일 생성
- 1차 버전에서는 데이터 결과만 정리해서 제공
- 추후 원하면 테마 분석 프롬프트 자동 첨부 기능 추가 가능

이 모듈은 공부용 / 리서치용 도구에 가깝습니다.

### Module 3. 개별 종목 분석

관심 종목 1개를 입력하면, 그 종목 분석용 데이터를 구조화해서 모아주는 기능입니다.

수집 대상 예시:

- 현재가, 등락률, 거래대금
- 시가총액, 52주 고저
- 이동평균선 위치
- 외국인 / 기관 수급 20일
- 재무 데이터
- 최근 공시
- 최근 뉴스
- 증권사 리포트 / 목표가
- 공매도 잔고
- 대주주 지분율
- 재무 위험 요소(CB/BW/유증 등)

출력 결과:

- 종목명 기준 텍스트 파일 생성
- 1차 버전에서는 데이터 결과만 정리해서 제공
- 추후 원하면 개별 종목 분석 프롬프트 자동 첨부 기능 추가 가능

이 모듈은 Module 1 다음 우선순위입니다.

## 4. 개발자가 해야 하는 일

개발자는 아래 기능을 구현하면 됩니다.

1. 데이터 수집기 만들기
   - 네이버 증권, KRX, Yahoo Finance 크롤링
   - DART OpenAPI 연동
2. 수집한 데이터를 일정한 형식으로 정리하기
3. 각 모듈별 `.txt` 파일 자동 생성하기
4. 자동 실행 스케줄 만들기
   - 장 마감 후 자동 실행
5. 수동 실행 버튼 만들기
6. 결과를 화면 복사 및 `.txt` 다운로드 가능한 형태로 제공하기
7. 추후 원하면 프롬프트 자동 첨부 기능을 붙일 수 있게 구조화하기
8. 추후 사이트 구조 변경 시 수정 가능하게 구조화하기

권장 기술:

- Python
- `requests`, `BeautifulSoup`, 필요 시 `Selenium`
- `FinanceDataReader`, `pykrx`
- `yfinance`
- `schedule`
- UI는 `tkinter`, `Flask`, `Streamlit` 중 택1

## 5. 당신이 해야 할 일

이 명세서를 받은 사람 입장에서 실제로 해야 할 일만 추리면 아래입니다.

### 꼭 준비해야 하는 것

1. DART OpenAPI 키 발급
   - 무료
2. 결과 파일을 어디에 저장할지 폴더 구조 결정
3. 자동 실행만 쓸지, 수동 버튼도 함께 쓸지 결정

### 개발자에게 명확히 말해줘야 하는 것

1. UI가 필요한지
   - 그냥 백그라운드 자동 실행만 할지
   - 버튼 있는 간단한 프로그램이 필요한지
2. 자동 실행 환경
   - 내 PC에서만 실행할지
   - 서버 없이 개인용으로만 쓸지
3. 출력 포맷 선호
   - `.txt`만 필요한지
   - 나중에 `.md`나 `.csv`도 필요한지
4. 추후 프롬프트 자동 첨부 기능이 필요한지
   - 1차 버전에서는 없어도 됨

### 실제 사용 시 당신이 하는 행동

1. 프로그램 실행 또는 자동 실행 대기
2. 생성된 텍스트 파일 열기
3. 전체 복사
4. ChatGPT/Gemini에 붙여넣기
5. AI 분석 결과 확인

---

## 6. 개발 우선순위

원문 기준 우선순위는 아래입니다.

1. Module 1 시황 브리핑
2. Module 3 개별 종목 분석
3. Module 2 테마 공부

이 순서가 맞는 이유:

- 시황 브리핑이 가장 자주 쓰임
- 구조상 Module 3과 재사용 가능성이 큼
- 테마 공부는 상대적으로 후순위로 빼기 쉬움

---

## 7. 주의사항

이 프로그램은 완전 자동 투자 프로그램이 아닙니다.

- 크롤링 대상 사이트 구조가 바뀌면 수정이 필요할 수 있음
- 일부 데이터는 장 마감 직후 바로 안 뜰 수 있음
- 증권사 리포트 전문 수집은 어려울 수 있음
- DART 데이터 일부는 지연 가능
- 최종 투자 판단은 사용자 책임
