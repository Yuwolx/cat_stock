# CAT STOCK × CAT COIN — 통합 확장 기획서

> **작성일**: 2026-05-29  
> **방향**: 기존 cat_stock 서비스에 코인 모드를 추가. 별도 서비스 아님.  
> **UX 원칙**: 좌측 상단 서비스 스위처로 주식 ↔ 코인 전환. 디폴트 주식.

## 3. 서비스 스위처 UX

- 헤더 좌측에 `CAT STOCK` / `CAT COIN` 두 버튼
- 현재 모드는 진하게(활성), 다른 쪽은 흐리게(비활성)
- 클릭 시 전체 탭·컨텐츠 교체
- **디폴트: STOCK**
- URL 쿼리 파라미터(`?mode=coin`)로 직링크 지원 고려 (선택)
- 각 모드의 session_state는 독립 (`stock_result`, `coin_result` 등)

## 4. 모드별 탭 구성

### STOCK 모드 (기존)

| 탭 | 입력 | 출력 |
|----|------|------|
| 시황 브리핑 | 기준일 날짜 | 지수·수급·공시·글로벌 매크로 |
| 개별 종목 분석 | 종목명 | DART 공시·재무요약·수급 |
| 테마 공부 | 테마명 | 관련 종목·뉴스·공시 |

### COIN 모드 (신규)

| 탭 | 입력 | 출력 |
|----|------|------|
| 코인 시황 | 기준 시각 (자동) | BTC/ETH·도미넌스·공포탐욕·업비트 상위·김치 프리미엄 |
| 개별 코인 | 코인명 (예: 비트코인, 솔라나) | 가격·시총·거래량·온체인 점수·주요 이슈 |
| 섹터 공부 | 섹터명 드롭다운 | 섹터 수익률·대표 코인·배경 |

## 6. 데이터 소스 (코인 모드)

| 소스 | 인증 | 데이터 |
|------|------|--------|
| CoinGecko API v3 | 무료, 키 불필요 | 가격·시총·거래량·도미넌스·점수·카테고리 |
| Upbit 공개 API | 무료, 키 불필요 | KRW 마켓 시세·거래대금 상위 |
| Alternative.me | 무료 | 공포탐욕지수 |
| FinanceDataReader | 기존 사용 중 | 달러인덱스·나스닥·미국채 (재사용) |

**김치 프리미엄** = (업비트 BTC 원화 ÷ (CoinGecko BTC USD × 달러원)) − 1

## 8. pages.py 변경 포인트

```python
def render_app() -> None:
    # 서비스 스위처: session_state로 mode 관리
    mode = st.session_state.get("mode", "stock")

    col_logo, col_center = st.columns([...])
    with col_logo:
        # CAT STOCK / CAT COIN 버튼
        if st.button("CAT STOCK", ...):
            st.session_state["mode"] = "stock"
        if st.button("CAT COIN", ...):
            st.session_state["mode"] = "coin"

    if mode == "stock":
        _render_stock_mode()   # 기존 3개 탭
    else:
        _render_coin_mode()    # 신규 3개 탭
```

## 11. 리스크

| 리스크 | 대응 |
|--------|------|
| CoinGecko 무료 Rate limit (30회/분) | `ThreadPoolExecutor` 병렬화 + `st.cache_data` 5분 캐싱 |
| 코인명 한글 입력 매핑 ("비트코인" → "bitcoin") | CoinGecko `/search` API + 자주 쓰는 코인 하드코딩 사전 |
| 코인은 24시간 거래 → "오늘 마감" 개념 없음 | 브리핑 기준을 현재 시각 스냅샷으로 고정 |
| 섹터 카테고리 명칭 영문 → 한글 표기 | 주요 섹터 한글 매핑 테이블 별도 관리 |
